## Operations

| | Mouse | Keyboard | Controller |
|---|---|---|---|
| Move | - | W/A/S/D | ◄/►/▲/▼, ⦿ |
| Look at | move | - | 〇 |
| Pick | MMB | R | A |
| Throw | RMB | F | Y, L |
| Attack | LMB | - | X, R |
